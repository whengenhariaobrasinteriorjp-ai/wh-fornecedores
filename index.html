🌐 Busca de Fornecedores Online

Para ativar a busca com IA, configure a chave de API da Anthropic no Netlify:
1. Acesse o painel do Netlify → seu site → Site configuration → Environment variables
2. Clique em "Add a variable"
3. Key: ANTHROPIC_API_KEY  |  Value: sua chave (começa com sk-ant-...)
4. Clique em Save → vá em Deploys → Trigger deploy
5. Pronto! A busca com IA estará funcionando.
💡 A chave fica segura no servidor Netlify — nunca aparece no navegador dos usuários. Obtenha sua chave em: console.anthropic.com

🤖 Buscar Fornecedor com IA + Web

Digite o nome ou tipo do fornecedor. A IA pesquisa na web, extrai os dados (telefone, e-mail, CNPJ, cidade) e adiciona diretamente à sua lista.

⚙️ CONFIGURAR BUSCA COM IA

Via Make.com — sem precisar de servidor ou configuração técnica.

PASSO A PASSO:
1. Acesse make.com → crie conta grátis
2. Crie um novo cenário → adicione módulo Webhooks → Custom webhook
3. Copie a URL do webhook gerada
4. Adicione módulo HTTP → Make a request com:
   • URL: https://api.anthropic.com/v1/messages
   • Method: POST
   • Headers: x-api-key + anthropic-version: 2023-06-01
   • Body: modelo + query do webhook
5. Adicione módulo Webhooks → Webhook response retornando o JSON
6. Cole a URL do webhook abaixo e salve
